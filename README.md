Skyslist - A realtime todo list app

- Write down todo items
- Work with other people in realtime
- Save to localStorage


To install:

```bash
git clone https://github.com/skysl/skyslist
cd skyslist
```

If you have `yarn`:

```bash
yarn install
```

If you just have `npm`:

```bash
npm install
```

And then to start the app:

```yarn start```

Or:

```npm start```
